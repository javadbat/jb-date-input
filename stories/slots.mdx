# Slots
jb-date-input allow you to use your own custom HTML inside certain position, for example you can change calendar icon or you can add some content in start or end of the input box.     

## Add Start Selection & End Section
You can easily add icon or any other HTML content before and after input inside input box by put `inline-start-section` or `inline-end-section` as slot attribute.    
here we put confetti icon with birthday text in `inline-start-section` slot and a start in the `inline-end-section` slot.
